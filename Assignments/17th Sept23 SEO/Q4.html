What happens when we try to write the css for the same element in different ways. like inline, internal and external ways.
Answer 2. when we target to style the same element using different ways then :
If styling using the inline styling approach then:
The inline styling has more specificity which means they will override the styles applied by the other ways.

If styling using the Internal styling approach then:
The internal styling has specificity between the inline and the external means, the style applied by the inline will get override by the inline style approach and internal styles will override to the styles applied by the external way.

If styling using the external styling approach then:
The external styling has less specificity. and the style applied by the inline and the internal ways will get the first preference over the external styling
