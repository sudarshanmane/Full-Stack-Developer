CSS var() Function

The css `var()` function is used to define and reference custom properties, also know as css variables. css variables allows to store and reuse values throughout the the stylesheet. CSS variables are defined using the --prefix followed by a variable name.

Purpose of var() Function
Reusability:
CSS variables allow us to store values that we can reuse across our stylesheet.
Consistency
by using variables, we can ensure consistent values for colors, spacing, fonts and more throughout our entire design .
Global Changes:
If we need to make a design change, we can update the variable's value once and changed will be applied to all
