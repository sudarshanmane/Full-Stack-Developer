Difference between justify-items and justify-self

In css, Justify-items and justify-self properties used to control the horizontal alignment of grid items within a grid container. Let's go over each property with code example.

1. Justify-items :

This property is used to set the default horizontal alignment for all grid items inside a grid container. It applies to all direct children of the grid container.
