z-index
About z-index
z-index property in css used to conrole the stacking order of the positioned elements. This property can applied to elements having "position" value other than 'static'

Explanation :

1. Elements with the higher z-index value are stacked above elements with lower value.
2. If two element have same z-index value, then the one that comes later will be stacked on the top.
3. In above example if we change the z-index of the second image to first it will stacked at the top and vise-versa
