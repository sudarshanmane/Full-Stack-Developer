The differences between HTML and HTML5:

HTML HTML5
Limited Semantic elements. challenging to define the structure of a document clearly. Introduced New semantic elements like <header>, <nav>, <main>. Now it is easy to define the structure of the document clearly.
Basic Form controls. with the limited input types and attributes. Introduced new input types and attributes. enhancing the form capabilities.
Limited Multimedia supports, required third party plugins for advanced multimedia content. Introduced the support for the multimedia like video and audio. No need to of the third party plugins.
