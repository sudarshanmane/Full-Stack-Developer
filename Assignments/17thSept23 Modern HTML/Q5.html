HTML5 Features
1. Semantic Elements
header, nav, aside, main, section, article, footer these elements helps to define the structure of the webpage more semantically. making it clearer and more meaningful.
2. Media Elements
<audio>, <video> these multimedia elements allow you to embed audio and video directly into your web pages.
3. Form Controls
1. <input> attributes like 'type'='email', 'type'='url', 'type'='tel' and other introduced.
2. <datalist> data list allow to provide some predefined options for an <input> element
4. API's
1. Web storage localStorage and sessionStorage : provides a way to store the data to the client's machine
2. <datalist> data list allow to provide some predefined options for an <input> element
